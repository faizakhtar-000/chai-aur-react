## 💻 Getting Started

Clone the repository:

```bash
git clone https://github.com/faizakhtar-000/chai-aur-react.git
```

Move into the project directory:

```bash
cd chai-aur-react
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
